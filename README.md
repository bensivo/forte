# Forte

Forte is a personal knowledge-base CLI that offloads organization to Claude while keeping the vault as browsable markdown.

## Prerequisites

- [uv](https://docs.astral.sh/uv/)
- Python 3.11+

## Install

```
uv sync
```

## Run the CLI

```
uv run forte
```

At the current scaffold stage this prints `hello-world`.

## Run tests

```
uv run pytest
```

## Lint & format

```
uv run ruff check .
uv run ruff format .
```
